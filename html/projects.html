Antifreeze Memetic (https://github.com/oddloops/antifreeze-memetics)
A program developed for the a friend who is a PhD candidate in SUNY Stony Brook. It is a comprehensive data processing pipeline to read, sort, filter, and visualization batches of text files of freezing data. Utilized Jupyter Notebook and Python for normalization and visualization.

Aphelion (https://github.com/oddloops/Aphelion)
Aphelion is a solar system simulation that employs object-oriented programming principles built using Python. Utilizing Python's Matplotlib library for data visualization and Tkinter library for graphical user interface it simulates the eight knownmplanets' orbit around the Sun. By using data taken from multiple sources such as NASA and applying physic equations, the simulation accurately demonstrates the shape and speed of each planet's orbit.

Periodic (https://github.com/oddloops/Periodic)
Periodic is desktop application that is a functioning Periodic Table of Elements that displays the periodic table and each element's information. Written in C# with the UI developed using the Windows Presentation Foundation (WPF) subsystem, Periodic provides a stylish and convenient periodic table of elements. Data for the periodic table was given by the National Center for Biotechnology Information (2023). Periodic Table of Elements.

Sisyphus Cycle (https://github.com/oddloops/SisyphusCycle)
Sisyphus Cycle is a workout tracking website that records personal records for weightlifting exercises. Built using a frontend of HTML, CSS, JavaScript with a Node.js backend with MySql serving as my database, this application allows users to create an account and easily enter and update the exercise name, body part worked, weight lifted, rep and set amount, and the date achieved. This web application is hosted using Heroku and JawsDB (May encounter an error as resource allocation may be turned off for the web application).
